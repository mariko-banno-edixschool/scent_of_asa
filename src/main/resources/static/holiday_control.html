運用メモ

優先順位

店舗休業日は、ガイド個別の公開設定より優先される想定です。ここで休業にした日は予約サイト全体で受付停止にします。

今後の連携

Java 側では `store_holiday` テーブルを持ち、予約画面のカレンダー生成時にガイド在庫より先に閉店日判定をかける構成がおすすめです。
